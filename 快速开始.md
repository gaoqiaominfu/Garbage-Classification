## ⚙️ 配置步骤

### 1. 配置 TMDB API Key
编辑 `utils/config.js`:
```javascript
tmdb: {
  apiKey: '你的 TMDB API Key'  // 在这里填入
}
```

**TMDB API Key 申请**:
1. 访问 https://www.themoviedb.org/
2. 注册账号
3. 设置 → API → 申请 Developer Key
4. 复制 API Key 填入

### 2. 配置小程序 AppID
编辑 `project.config.json`:
```json
{
  "appid": "你的小程序 AppID"
}
```

## ⚠️ 常见问题

### Q: 搜索不到内容？
A: 检查 TMDB API Key 是否配置正确

### Q: 图片加载失败？
A: 在小程序后台配置域名：
- `api.themoviedb.org`
- `image.tmdb.org`

### Q: 豆瓣 API 不稳定？
A: 正常，会自动切换到 TMDB
