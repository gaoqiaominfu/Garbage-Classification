# 🔧 搜索没反应？快速诊断

## ⚡ 3 步诊断

### 第 1 步：检查 TMDB API Key

**打开 `utils/config.js`**

```javascript
tmdb: {
  apiKey: '' // ⬅️ 这里是空的吗？
}
```

**如果是空的 → 去申请！**
```
https://www.themoviedb.org/settings/api
```

**填入你的 API Key：**
```javascript
apiKey: '你的 API Key',
```

---

### 第 2 步：检查域名配置

**小程序后台** → 开发管理 → 开发设置 → 服务器域名

**request 合法域名 必须有：**
```
https://api.themoviedb.org
https://image.tmdb.org
```

**没有就添加！**

---

### 第 3 步：查看控制台错误

**微信开发者工具** → 编译 → 点击搜索 → 看控制台

**常见错误：**

| 错误信息 | 原因 | 解决 |
|---------|------|------|
| `apiKey is required` | 没填 API Key | 填入 TMDB API Key |
| `url not in domain list` | 域名没配置 | 小程序后台加域名 |
| `request:fail` | 网络问题 | 检查网络/代理 |
| 没错误但没结果 | API Key 不对 | 重新申请 API Key |

---

## 🧪 测试方法

### 方法 1: 用测试脚本

**打开 `test-tmdb.js`**

1. 填入你的 API Key
2. 复制到微信开发者工具控制台
3. 运行 `testSearch('狂飙')`
4. 看输出

**成功输出：**
```
✅ TMDB API 配置正确！可以正常使用！
📺 找到影视作品：10
```

**失败输出：**
```
❌ API 返回错误：{"status_code":3,"status_message":"Invalid API key"}
```
→ API Key 无效，重新申请！

---

### 方法 2: 直接搜索

1. 微信开发者工具 → 编译
2. 进入搜索页
3. 输入「狂飙」
4. 点击搜索
5. 看控制台和结果

## 🐛 还是不行？

### 终极解决方案：

**1. 清空缓存**
```
微信开发者工具 → 工具 → 清除缓存 → 全部清除
```

**2. 重新编译**
```
点击「编译」
```

**3. 真机预览**
```
点击「预览」→ 手机扫码
```

**4. 查看日志**
```
微信开发者工具 → 调试器 → Console
```
