# 🌩️ 云函数部署指南

## 📦 云函数说明

**函数名**: `doubanSearch`  
**功能**: 爬取豆瓣移动搜索页，返回影视数据  
**优势**: 无需 API Key，中文数据准确

---

## 🚀 部署步骤

### 步骤 1: 开通云开发

1. 打开**微信开发者工具**
2. 点击工具栏「云开发」按钮
3. 开通云开发环境 (免费版即可)
4. 记录**环境 ID**

---

### 步骤 2: 配置云函数

编辑 `app.js`:
```javascript
// app.js
App({
  onLaunch: function () {
    if (!wx.cloud) {
      console.error('请使用 2.2.3 或以上版本，以支持云开发能力');
    } else {
      wx.cloud.init({
        env: 'your-env-id',  // 填入你的云环境 ID
        traceUser: true,
      });
    }
  }

```

### 步骤 5: 小程序端调用

编辑 `pages/search/search.js`:
```javascript
// 使用云函数搜索
searchByCloud(keyword) {
  wx.showLoading({ title: '搜索中...' });
  
  wx.cloud.callFunction({
    name: 'doubanSearch',
    data: {
      keyword: keyword,
      type: 'tv'  // 或 'movie'
    }
  }).then(res => {
    wx.hideLoading();
    
    if (res.result.code === 200) {
      const results = res.result.data.results;
      
      this.setData({
        searchResult: results,
        loading: false,
        apiSource: 'douban-cloud'
      });
      
      if (results.length === 0) {
        wx.showToast({
          title: '未找到相关内容',
          icon: 'none'
        });
      } else {
        wx.showToast({
          title: `找到${results.length}条结果`,
          icon: 'success'
        });
      }
    } else {
      wx.showToast({
        title: res.result.message || '搜索失败',
        icon: 'none'
      });
    }
  }).catch(err => {
    wx.hideLoading();
    wx.showToast({
      title: '云函数调用失败',
      icon: 'none'
    });
  });
}
```

## ⚠️ 注意事项

### 1. 域名配置
在小程序后台添加合法域名：
```
request 合法域名：
https://m.douban.com
```

### 2. 频率限制
- 豆瓣：同一 IP 频繁请求会被限流
- 解决：云函数加缓存 (云数据库)

### 3. 数据缓存
建议缓存搜索结果：
```javascript
// 云函数中
const db = cloud.database();
const cache = await db.collection('search_cache')
  .where({ keyword, type })
  .get();

if (cache.data.length > 0) {
  return cache.data[0];  // 返回缓存
}
```

## 🐛 常见问题

### Q: 云函数调用超时？
A: 增加超时配置到 10 秒

### Q: 搜索结果为空？
A: 检查豆瓣页面结构是否变化，更新解析逻辑

### Q: 频繁被限流？
A: 加缓存 + 降低请求频率

### Q: 云函数部署失败？
A: 检查 `package.json` 依赖是否完整
