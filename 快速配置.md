## 方案 1: 云函数 (最推荐⭐)

**优点**: 无需 API Key，中文数据准确，稳定

### 步骤:

**1. 开通云开发**
- 微信开发者工具 → 云开发
- 开通免费版环境

**2. 上传云函数**
```bash
# 在微信开发者工具中
右键 cloudfunctions/doubanSearch
→ 上传并部署：云端安装依赖
```

**3. 配置 app.js**
```javascript
// app.js
wx.cloud.init({
  env: 'your-env-id'  // 填入你的云环境 ID
});
```

**4. 配置域名**
小程序后台 → 开发管理 → 开发设置：
```
request 合法域名：
https://m.douban.com
```

**5. 测试**
```javascript
// 控制台测试
wx.cloud.callFunction({
  name: 'doubanSearch',
  data: { keyword: '狂飙', type: 'tv' }
}).then(console.log)
```

## 方案 2: TMDB API

**优点**: 稳定可靠，数据全

### 步骤:

**1. 申请 API Key**
1. https://www.themoviedb.org/
2. 注册账号
3. 设置 → API → 申请 Developer Key
4. 复制 API Key

**2. 配置 utils/config.js**
```javascript
tmdb: {
  apiKey: '你的 API Key'  // 粘贴这里
}
```

**3. 配置域名**
小程序后台添加：
```
request 合法域名：
https://api.themoviedb.org
https://image.tmdb.org
```

**4. 测试搜索**
编译 → 搜索"流浪地球"

## 方案 3: 混合模式 (最佳实践)

**自动切换**: 云函数 → TMDB → 豆瓣代理

### 配置:

**utils/config.js**
```javascript
apiProvider: 'auto'  // 自动切换
```

**pages/search/search.js** (已配置)
- 优先云函数
- 失败降级到 TMDB
- 再失败用豆瓣代理

## 🐛 常见问题

### Q: 搜索不到内容？
**A**: 
1. 检查是否配置了数据源
2. 查看控制台错误信息
3. 检查域名是否配置

### Q: 云函数调用失败？
**A**:
1. 检查云开发是否开通
2. 检查环境 ID 是否正确
3. 查看云函数日志

### Q: 图片加载失败？
**A**:
1. 小程序后台配置域名
2. 检查图片 URL 是否正确
