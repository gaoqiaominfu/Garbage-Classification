# UI 设计说明

## 设计风格

### 整体风格
- **现代简约**：干净的界面，清晰的层次，留白充足
- **卡片式布局**：内容模块化，易于浏览和操作
- **视觉层次**：通过阴影、颜色、大小区分重要性

## 交互动效

### 点击反馈
- 卡片：scale(0.98) + 阴影加深
- 按钮：背景色变深
- 标签：颜色反转

### 过渡动画
- 进度条：width 0.3s ease
- 卡片：all 0.3s ease
- 主题切换：all 0.3s ease

### 加载状态
- 骨架屏：灰色闪烁动画
- Loading：文字 + 旋转图标

## 响应式适配

### 不同屏幕尺寸
- 使用 rpx 单位自动适配
- 最小适配宽度：320px
- 最大适配宽度：750px

### 安全区域
```css
padding-bottom: constant(safe-area-inset-bottom);
padding-bottom: env(safe-area-inset-bottom);
```

## 无障碍设计

- 文字对比度 ≥ 4.5:1
- 可点击区域 ≥ 44px × 44px
- 支持系统字体大小设置
- 图片添加 alt 描述

## 设计原则

1. **一致性**：相同功能的元素保持统一样式
2. **简洁性**：去除不必要的装饰，突出内容
3. **可用性**：操作直观，反馈及时
4. **美观性**：色彩和谐，布局平衡
5. **性能**：减少复杂动画，保证流畅度
