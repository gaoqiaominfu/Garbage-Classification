# 豆瓣爬虫方案说明

## ⚠️ 问题说明

小程序端**不能直接爬网页**，原因：
1. **CORS 限制** - 小程序只能请求配置的域名
2. **域名白名单** - 需在小程序后台配置 `m.douban.com`
3. **HTML 解析困难** - 小程序没有 DOM API

## ✅ 可行方案

### 方案 1: 云函数代理 (推荐)

**架构**:
```
小程序 → 云函数 (Node.js) → 豆瓣网页 → 返回 JSON → 小程序
```

**优点**:
- ✅ 无 CORS 限制
- ✅ 可解析 HTML
- ✅ 稳定可靠

**缺点**:
- ❌ 需开通微信云开发
- ❌ 有免费额度限制

---

### 方案 2: TMDB 为主 + 手动输入 (最简单)

**架构**:
```
小程序 → TMDB API → 返回数据
```

**优点**:
- ✅ 无需爬虫
- ✅ 稳定可靠
- ✅ 数据全

**缺点**:
- ❌ 需申请 API Key
- ❌ 中文数据不如豆瓣

---

### 方案 3: 自建代理服务器

**架构**:
```
小程序 → 自家服务器 → 豆瓣网页 → 返回 JSON
```

**优点**:
- ✅ 完全控制
- ✅ 可缓存数据

**缺点**:
- ❌ 需服务器
- ❌ 需维护

## 📝 TMDB API Key 申请步骤

1. **访问**: https://www.themoviedb.org/
2. **注册**: 点击 "Join TMDB"
3. **申请 API**:
   - 设置 → API → 申请 Developer Key
   - 填写应用信息 (个人学习用)
4. **复制 API Key** 到 `utils/config.js`

```javascript
// utils/config.js
tmdb: {
  apiKey: '你的 API Key'  // 粘贴这里
}
```

## 🚀 云函数方案代码

如果皇上选云函数，太子已准备好代码：

### 云函数：doubanSearch/index.js
```javascript
const cloud = require('wx-server-sdk');
const axios = require('axios');
const cheerio = require('cheerio');

cloud.init();

exports.main = async (event, context) => {
  const { keyword, type = 'tv' } = event;
  
  try {
    // 爬豆瓣移动搜索页
    const url = `https://m.douban.com/search/`;
    const response = await axios.get(url, {
      params: { query: keyword, type },
      headers: {
        'User-Agent': 'Mozilla/5.0 (iPhone; CPU iPhone OS 14_0 like Mac OS X)'
      }
    });
    
    // 解析 HTML
    const $ = cheerio.load(response.data);
    const results = [];
    
    $('.card').each((i, elem) => {
      const title = $(elem).find('.title').text().trim();
      const rating = $(elem).find('.rating').text();
      const image = $(elem).find('img').attr('src');
      const year = $(elem).find('.year').text();
      
      if (title) {
        results.push({
          title,
          rating,
          image,
          year,
          type
        });
      }
    });
    
    return {
      code: 200,
      data: { results }
    };
  } catch (error) {
    return {
      code: 500,
      message: error.message
    };
  }
};
```
