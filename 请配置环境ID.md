# ⚠️ 重要：请配置云开发环境 ID

## 📍 步骤

### 1. 获取环境 ID

**微信开发者工具** → 点击顶部「云开发」☁️ → 左上角显示的就是环境 ID

格式类似：`cloud1-xxx` 或 `prod-xxx`

点击环境 ID 即可复制！

---

### 2. 填入 app.js

打开 `app.js`，找到第 7 行：

```javascript
wx.cloud.init({
  env: 'your-env-id',  // ⬅️ 改成你的环境 ID
  traceUser: true,
});
```

**改为**:

```javascript
wx.cloud.init({
  env: 'cloud1-abc123',  // ⬅️ 填入你复制的环境 ID
  traceUser: true,
});
```

---

### 3. 保存并编译

- 保存 `app.js`
- 点击「编译」
- 查看控制台是否显示：`✅ 云开发已初始化`

---

### 4. 上传云函数

**右键** `cloudfunctions/doubanSearch` 文件夹  
→ **上传并部署：云端安装依赖**

---

### 5. 测试搜索

1. 编译小程序
2. 进入搜索页
3. 搜索「狂飙」
4. 应该能看到搜索结果！

---

## 🐛 常见问题

### Q: 没有云开发按钮？
**A**: 
1. 确认微信开发者工具版本 ≥ 2.2.3
2. 项目需关联小程序

### Q: 云开发控制台打不开？
**A**: 先开通云开发（免费版即可）

### Q: 有多个环境选哪个？
**A**: 选第一个或最新的

---
